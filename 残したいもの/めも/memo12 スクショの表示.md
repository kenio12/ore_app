スクリーンショットの表示パターン（最終版）

スクショは２枚まで載せれることにしよう。

1. スマホアプリの場合（2枚）
PCで見る時（横に2枚）：
┌──┐ ┌──┐
│ス│ │ス│
│マ│ │マ│
│ホ│ │ホ│
│画│ │画│
│面│ │面│
└──┘ └──┘

スマホで見る時（縦に2枚）：
┌──┐
│ス│
│マ│
│ホ│
└──┘
  ↓
┌──┐
│ス│
│マ│
│ホ│
└──┘

2.デスクトップアプリの場合（1枚）
PCで見る時（横に2枚）：
┌────────┐ ┌────────┐
│  PC画面1  │ │  PC画面2  │
│(横長画像) │ │(横長画像) │
└────────┘ └────────┘

スマホで見る時（縦に2枚）：
┌────────┐
│  PC画面1  │
│(横長画像) │
└────────┘
      ↓
┌────────┐
│  PC画面2  │
│(横長画像) │
└────────┘

3.Webアプリの場合（2枚）
PCで見る時（横並び）：
┌────────┐  ┌──┐
│  PCビュー │  │ス│
│ (横長画像)│  │マ│
└────────┘  │ホ│
             └──┘

スマホで見る時（縦並び）：
┌────────┐
│  PCビュー │
│(横長画像) │
└────────┘
     ↓
┌──┐
│ス│
│マ│
│ホ│
└──┘

全パターンで2枚に統一
PCで見た時のカードサイズが揃う
レイアウトの一貫性が保てる
スマホでも見やすい
